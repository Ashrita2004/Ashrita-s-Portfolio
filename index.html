Algorithm-Pathfinder

This JavaFX Pathfinding Visualizer provides an interactive and analytical demonstration of DFS, BFS, and Dijkstra’s algorithms. The application features real-time animations that illustrate how each algorithm explores the grid, processes weighted obstacles, and determines the shortest path. To support deeper analysis, it includes an integrated metrics panel and dynamic charts that present key performance indicators such as visited nodes, execution time, and path cost.
Tech Stack: Java, JavaFX (LineChart, BarChart, Timeline), Stack, Queue, PriorityQueue.
